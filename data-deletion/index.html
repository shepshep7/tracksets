Data deletion

Delete Track Sets data

Track Sets does not create user accounts and does not transmit exercise data to a server. There is no cloud account or server-side data for the developer to delete.

Delete local data

  • Delete individual exercises inside Track Sets.
  • Clear Track Sets storage in your device settings.
  • Uninstall Track Sets to remove its local app data from the device.
